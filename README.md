# routemap
> Simple Map that Displays a Route, its Stops and its Buses.

## demo
https://gocarta.github.io/routemap/?route=1

## usage
You can easily map a route and its stops by setting the route parameter in the URL.  Here's some more examples:
- https://gocarta.github.io/routemap/?route=4
- https://gocarta.github.io/routemap/?route=15
- https://gocarta.github.io/routemap/?route=33
